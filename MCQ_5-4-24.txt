1. Which of the following methods can be used to sort an array in Java?
(a) Arrays.sort();
(b) Collections.sort();
(c) List.sort();
(d) ArrayList.sort();
Answer: (a)
2. Which of the following methods can be used to search for an element in an array in Java?
(a) Arrays.search();
(b) Collections.search();
(c) List.search();
(d) ArrayList.search();
Answer: (a)
3. Which of the following statements is true about multidimensional arrays in Java?**
(a) Multidimensional arrays are arrays of arrays.
(b) Multidimensional arrays can only store primitive data types.
(c) Multidimensional arrays can only store reference data types.
(d) Multidimensional arrays are immutable.
Answer: (a)
4. Which of the following is the correct way to declare a two-dimensional array of integers in Java?*
(a) int arr[][] = new int;
(b) int[] arr[] = new int;
(c) int[][] arr = new int;
(d) int arr = new int();
Answer: (c)
5. Which of the following is the correct way to access the element at index (2, 3) of a two-dimensional array named arr?*
(a) arr;
(b) arr(2, 3);
(c) arr.2.3;
(d) arr->2->3;
Answer: (a)
